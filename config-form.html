Home Cinema Config Form

Runs entirely in your browser. It generates local config text; nothing is uploaded anywhere.

GitHub safety: commit this form and the examples, but do not commit generated files containing tokens.
Remote tunnel note: if you paste a Cloudflare Tunnel token, the installer connects this Debian server to an existing Cloudflare Tunnel. It does not publish public hostnames. Add them in Cloudflare Zero Trust, or leave the token blank and use the locally managed tunnel flow.
